## 📖 About the Project

**DisciplineForge** is a premium, highly-interactive habit and task tracker engineered to help you build unshakeable discipline. 

Moving away from the clutter of traditional to-do lists, DisciplineForge focuses on **streaks**, **consistency**, and **visual progression**. It features an incredibly sleek dark-mode aesthetic with modern glassmorphism, completely transforming into a native iOS-style application when viewed on mobile devices.

### ✨ Key Features

- **GitHub-Style Contribution Heatmaps:** Visualize your discipline over time. Every checkmark fuels your heatmap, giving you a beautiful visual representation of your consistency.
- **Dynamic Streak Tracking:** Track your current and longest streaks. Watch the fire emoji 🔥 ignite as your current streak grows!
- **Granular Subtasks:** Break massive goals into actionable chunks. A task is only complete when all subtasks are finished.
- **Native iOS Mobile Experience:** A fully responsive design that transforms the top navigation into a sleek bottom tab bar on mobile screens, making it feel like a true native app.
- **Seamless Cloud Syncing:** Built on a robust cloud architecture. Start tracking on your laptop and continue flawlessly on your mobile phone.
- **One-Click Google Login:** Instant, secure authentication without the hassle of remembering passwords.

## 🛠 Tech Stack & Architecture

DisciplineForge was built focusing on maximum performance and zero bloat. 

- **Frontend Core:** Vanilla JavaScript (ES6+), HTML5, CSS3
- **Styling:** Custom CSS implementing a premium Glassmorphism design system. (No heavy CSS frameworks like Tailwind or Bootstrap were used, ensuring lightning-fast load times).
- **Build Tool:** [Vite](https://vitejs.dev/) - For ultra-fast hot module replacement and optimized production bundling.
- **Backend & Database:** [Google Firebase](https://firebase.google.com/)
  - **Firebase Auth:** Handles secure user sessions and Google OAuth integration.
  - **Cloud Firestore:** A NoSQL cloud database providing real-time synchronization of tasks, subtasks, and streaks across all user devices.
- **Deployment:** [Vercel](https://vercel.com/) - Providing edge-network hosting for instant load speeds globally.

## 🚀 Experience the App

You can use the fully functional live application here:  
👉 **[staydiscipline.vercel.app](https://staydiscipline.vercel.app/)**

### How it works:
1. **Sign Up:** Use the secure email/password form or click **Continue with Google**.
2. **Forge a Habit:** Click the floating green `+` button to create a new task (e.g., "Gym").
3. **Add Subtasks:** Break it down (e.g., "30m Cardio", "Protein Shake").
4. **Execute:** Check them off daily. Watch your heatmap light up green and your discipline score skyrocket!

## 💻 Local Development

Want to run DisciplineForge on your local machine?

1. **Clone the repository**
   ```bash
   git clone https://github.com/r-samarth/discipline-forge.git
   cd discipline-forge
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
   The app will be running at `http://localhost:3000`.
